Withholding Tax

Withholding Tax

This module is used for the setting and calculation of Withholding Tax (WHT).
It enables user to set up WHT as per the regulation of their respective region of operation.

Description

Description

Tax withholding, also known as tax retention, Pay-as-You-Go, Pay-as-You-Earn, is income tax paid to the government by the payer of the income rather than by the recipient of the income. The tax is thus withheld or deducted from the income due to the recipient

Its calculation can be tricky at times and needs to be handled delicately. This module helps make that job easy.

Standout Features

Standout Features

  • The default WHT rate can be set and Overridden.
  • It gives the option to set it to
    • Sales
    • Purchases and
    • WHT on Taxes, separately.

Exploring the module: (How to)

Exploring the module: (How to)

Once the module is integrated to your system,open the Invoicing App.

  • Download and Install the module
  • Open the Invoicing module. Click on the Configuration tab from the menu. The configuration tab provides a convenient hub to the different parts of the Invoicing module.
  • Creating a WHT
    • Select the Withholding Tax option from the menu.
    • A list of already created taxes appears with details like its name, code, scope and description. These can be Activated or deactivated as required.
    • To create a new WHT click on the Create button on the top left corner and a form will appear. The list can be exported in a csv or xls format as well.
  • Enter the details in the relevant fields like
    • Tax name.
    • Relevant Tax code
    • Description (if required)
    • Tax Scope: A dropdown menu appears which enables you to choose whether the tax is on
      • Sales
      • Purchases
      • Or WHT on Taxes withheld.
      • Tax application method can be chosen whether it is applied At Payment or at Invoice stage.
      • You can choose to activate or deactivate the tax by ticking or unticking the next box.
      • Next in the definition tab enter the accounting details like tax rate, WHT Account associated (from the companyâ€™s chart of Accounts) and WHT Account on Credit Notes.
      • In the Advance Options tab you can select a predefined Tax Group or you can define a new one by clicking this option.
      • The following form appears. Enter the relevant details to define a new group.
  • The functionality helps to set up WHT on the Vendor bills, customer Invoices, and Payments.
  • Vendor Bills: Go to Vendor menu select Bills, enter the relevant details in the form appearing and then add a line section you can select the WHT rate applicable.
  • Customer Invoices: Customer Invoices and its WHT can be setup in pretty much the same way.
  • Customer Payments: In the customer menu select payments and set the WHT in the given form.
  • Automatic WHT calculation and display: The module calculates the WHT from and displays the info. Like the WHT rates, the tax amount and the net amount.
  • OVERRIDING THE WHT: This is the feature that makes this module stand out. It enables the user to override the WHT set and change the rate if they want to.
    • Go to the Vendors menu, press products, click on Edit and then select Purchases tab. In the Vendor Taxes tab you can override the WHT selected.
  • Setting up WHT on Invoices for Customers: Go to Customers menu and set up WHT as in the vendor menu above. To select one on invoice. Go to customers then invoices, in the Add a Line tab, select add a line and in the WHT column select the tax already setup.
